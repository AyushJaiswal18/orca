# Server Documentation

Backend server documentation for the Express.js API.

## Overview

The server is built with Express.js and includes:
- MongoDB with Mongoose
- JWT Authentication
- BullMQ job queues
- Cloudinary file uploads
- Razorpay payments
- Resend email service
- Winston logging
- Prometheus metrics
- Cron jobs

## Quick Start

```bash
cd server
npm install
npm run dev
```
